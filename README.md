# uju

A 3D space flight simulator written in C/C++ using Raylib.

## What does `uju` stand for?

Uju (우주) is the Korean word for space/outerspace.

## Pre-requisites

- CMake

## Building

```bash
mkdir build
cd build
cmake ..
cmake --build .
```
